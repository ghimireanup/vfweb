# Aura Landing Page

This is a modern, responsive landing page template designed for SaaS products.

## How to use

1.  **Open locally**: simply double-click `index.html` to view the page in your browser.
2.  **Upload to GitHub**:
    *   Create a new repository on GitHub.
    *   Run the following commands in this folder:
        ```bash
        git init
        git add .
        git commit -m "Initial commit"
        git branch -M main
        git remote add origin <your-repo-url>
        git push -u origin main
        ```
    *   To publish, go to your repository Settings > Pages and select the `main` branch as the source.

## Tech Stack
*   HTML5
*   CSS3 (Variables, Flexbox, Grid, Glassmorphism)
*   Vanilla JavaScript
